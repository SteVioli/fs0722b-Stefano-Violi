HTML BASE

Elementi PRINCIPALI HTML

  • !DOCTYPE html Rappresenta il tipo di documento
  • HTML Elemento radice della pagina HTML
  • TAG HEAD Contiene l'informazione strutturale della pagina
  • TITLE Titolo della pagina, serve a noi per dare una semantica corretta di quello che stiamo facendo
  • TAG HEADER Nella nostra pagina serve a dividere per importanza il contenuto, è la parte superiore (scorrimento fisso menù)
  • TAG MAIN Parte centrale che mantiene l'argomento principale della pagina per semantica
  • TAG FOOTER Parte inferiore della pagina che può contenere contatti, link
  • TAG NAV Tag che bisogna usare per contenere il menu dei link di NAVIGAZIONE della pagina
  • TAG SECTION Tag da utilizzare per dividere i vari contenuti della pagina a seconda dei vari argomenti
  • TAG da h1 a h6 Sono i vari titoli di importanza della pagina (in ordine di importanza h1->h6)
  • TAG p Paragrafo
  • TAG div Sono i nostri CONTENITORI
  • TAG br Serve per andare a capo
  • TAG span Sezione inline
  • COMMENTOctrl + c + v
  • TAG abbr Abbreviazione
  • TAG ol Lista ordinata
  • TAG ul Lista non ordinata
  • TAG li Oggetto della lista
  • TAG table Definisce una tabella
  • TAG td Cella della tabella
  • TAG tr Riga tabella
  • TAG th Cella di testa in una tabella
  • TAG theadContenuto di testa in una tabella
  • TAG tbodyContenuto del corpo di una tabella
  • TAG tfootContenuto footer di una tabella
